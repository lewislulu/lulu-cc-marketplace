# lulu-cc-marketplace

Lewis's curated collection of Claude Code plugins.

## Install

In Claude Code, run:

```
/plugin marketplace add https://github.com/lewislulu/lulu-cc-marketplace
```

## Plugins

### 🎨 html-ppt

Author professional HTML presentations with AI. One skill, 100+ PPT capabilities.

- **36 themes** — from minimal-white to cyberpunk-neon
- **31 layouts** — KPI grids, timelines, architecture diagrams, and more
- **47 animations** — 27 CSS transitions + 20 canvas particle FX
- **14 full-deck templates** — tech sharing, pitch deck, weekly report, 小红书图文, etc.

```
npx skills add github.com/lewislulu/html-ppt-skill
```

> Also available as standalone: [html-ppt-skill](https://github.com/lewislulu/html-ppt-skill)
